# Portfolio Website

A modern, professional portfolio website showcasing GitHub projects with video demonstrations.

## Setup

1. **Add your videos**: Place your project demo videos in a `videos/` folder
   - Recommended format: MP4 and WebM for browser compatibility
   - Optimal size: 1920x1080 or 1280x720
   - Keep file sizes reasonable (under 5MB per video)

2. **Update content**: Edit `index.html` to add your projects
   - Replace GitHub URLs
   - Update project titles and descriptions
   - Add appropriate tags
   - Update footer links and contact info

3. **Customize styling**: Modify `styles.css` if you want to adjust colors or layout
   - Color scheme is defined in CSS variables at the top
   - Easy to switch to light mode or different accent colors

## Features

- Responsive grid layout
- Auto-playing video loops on hover
- Smooth animations and transitions
- Lazy loading for performance
- Accessibility compliant
- Mobile-friendly design
- Subtle film grain texture
- Parallax header effect

## Video Recording Tips

For best results when recording your tool demos:
- Use screen recording software like OBS, QuickTime, or Windows Game Bar
- Record at 30fps minimum
- Keep videos short (5-15 seconds loops work best)
- Show the most impressive features
- Use a clean, uncluttered screen
- Consider adding subtle zoom or pan for visual interest

## Deployment

Deploy to any static hosting service:
- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages

Just upload all files including the `videos/` folder.
